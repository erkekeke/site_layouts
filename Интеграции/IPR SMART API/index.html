IPR SMART API

IPR SMART API — сервис, специально разработанный Компанией IPR MEDIA для интеграции ЭИОС вуза с электронно-библиотечной системой IPR SMART.

Мы постарались сделать сервис максимально удобным для интеграции:

  • вы можете получить списки доступных публикаций каталогов книг  и периодики;
  • управлять пользователями IPR SMART;
  • просматривать статистические отчёты книговыдач и скачиваний изданий для offline-чтения, а также работы пользователей на мобильных устройствах.

Для подключения к сервису Вам необходимы 2 параметра,
которые вы передадите в отдел разработки программного
обеспечения вашей организации:

ВАМ СТАНУТ ДОСТУПНЫ

  • Кнопка «Войти в IPR SMART», при клике на которую пользователь портала или АСУ перейдёт в ЭБС уже авторизованным пользователем
  • Каталог изданий IPR SMART с возможностью перехода к изданиям в ЭБС (для авторизированных пользователей)
  • Книгообеспеченность по дисциплинам университета (из модуля книгообеспеченность)
