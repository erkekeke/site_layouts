Мобильное приложение для лиц с ОВЗ по зрению IPRbooks WV-Reader

Мобильное приложение IPRbooks WV-Reader позволяет слабовидящим и полностью незрячим людям комфортно работать с ЭБС IPR SMART на смартфоне или планшете с операционными системами IOS или Android.

Приложение прошло экспертизу Института «Реакомп» Всероссийского общества слепых, получило положительное заключение и рекомендацию к использованию. А также рекомендовано специалистами лаборатории технических и программных

Функциональные возможности WV-Reader

  • Адаптированный в соответствии с ГОСТом интерфейс
  • Запуск и работа при помощи голосовых команд
  • Удобная навигация по содержанию
  • Голосовой поиск изданий и голосовые ответы на запросы
  • Синтезатор речи, позволяющий слушать найденное издание
Издания адаптируются в приложении с помощью приемов тифлокомметирования: лаконичное описание предмета, пространства или действия, включая описание рисунков, формул, таблиц и схем, которые непонятны незрячему человеку без специальных словесных пояснений.

Каталог изданий приложения IPRbooks WV-Reader

Работа с приложением не требует специально обученного ассистента. Пользователи с ограниченными возможностями по зрению используют смартфон или планшет привычным способом с помощью штатных сервисов для незрячих людей.

Использование учебными заведениями по IPRbooks WV-reader позволит получить документальное подтверждение выполнения требований законодательства в части внедрения в учебный процесс адаптивных технологий для лиц с ОВЗ, что необходимо в том числе в целях заполнения форм ежегодного мониторинга деятельности образовательных организаций высшего образования.

Программное обеспечение подходит для людей, имеющих I, II, III группы инвалидности по зрению.
